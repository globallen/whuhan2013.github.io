**碰到的问题** 

- react-native可能在cmd窗口提示不是内部或外部命令            
解决方法：在nodeJS command prompt下可以运行 
- 运行时卡在最后，程序是白屏 
解决方法：为应用程序添加悬浮窗权限  
- 下载时卡住了 
解决方法: 更换rpm的数据源，用国内代理
